# problem-solving

### 레포지토리 구조
```bash
algorithm-repo/
│
├── README.md                       # 레포지토리 개요 및 목차
├── docs/                           # 알고리즘, 자료구조 개념 및 전략 문서 폴더
│   ├── algorithms/                 # 알고리즘 개념 정리
│   │   ├── sorting.md              # 예: 정렬 알고리즘 개념
│   │   ├── searching.md            # 예: 탐색 알고리즘 개념
│   │   └── dynamic_programming.md  # 예: 동적 계획법 개념
│   │
│   ├── data_structures/            # 자료구조 개념 정리
│   │   ├── arrays.md               # 예: 배열 개념
│   │   ├── linked_lists.md         # 예: 연결 리스트 개념
│   │   └── trees.md                # 예: 트리 개념
│   │
│   ├── strategies/                 # 문제풀이 전략 정리
│   │   ├── greedy.md               # 예: 그리디 알고리즘 전략
│   │   ├── divide_and_conquer.md   # 예: 분할 정복 전략
│   │   └── backtracking.md         # 예: 백트래킹 전략
│   │
│   └── README.md                   # docs 폴더 개요 및 목차
│
├── problems/                       # 문제 풀이 코드 저장 폴더
│   ├── platform_name/              # 플랫폼별 폴더 (예: LeetCode, Baekjoon)
│   │   ├── easy/                   # 난이도별 분류 (예: easy, medium, hard)
│   │   │   ├── problem1.py         # 문제 코드 파일
│   │   │   └── problem1.md         # 문제 설명 및 풀이 문서
│   │   └── medium/
│   │       ├── problem2.py
│   │       └── problem2.md
│   │
│   └── README.md                   # problems 폴더 개요 및 목차
│
└── resources/                      # 참고 자료 및 유용한 링크 모음
    ├── books.md                    # 알고리즘 관련 책 리스트
    ├── websites.md                 # 유용한 웹사이트 및 강의
    └── tools.md                    # 추천하는 도구 및 플러그인

```
