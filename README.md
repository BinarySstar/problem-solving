# problem-solving

### 레포지토리 구조
```bash
problem-solving/
│
├── README.md                       # 레포지토리 개요 및 목차
├── docs/                           # 알고리즘, 자료구조 개념 및 전략 문서 폴더
│   ├── algorithms/                 # 알고리즘 개념 정리
│   ├── data_structures/            # 자료구조 개념 정리
│   ├── strategies/                 # 문제풀이 전략 정리
├── problems/                       # 문제 풀이 코드 저장 폴더
│   ├── platform_name/              # 플랫폼별 폴더 (예: Programmers, Baekjoon)
│   │   ├── easy/                   # 난이도별 분류 (예: easy, medium, hard, 레벨)
│   │   └── medium/
└── resources/                      # 참고 자료 및 유용한 링크 모음
    └── README.md                  
```
